# TAIGA IACT Preprocessing Course

Учебный мини-проект по анализу данных TAIGA IACT для задачи классификации событий `gamma` и `proton`. Репозиторий будет пополняться заданиями курса: от первичного анализа и выбора признаков до обучения и сравнения моделей.

В репозитории собраны:

- CSV-данные с параметрами Хилласа для gamma- и proton-событий;
- `data/proton.csv` как подготовленная таблица proton-событий;
- PDF-справочник по параметрам TAIGA-IACT;
- учебный ноутбук `student_preprocessing_tasks.ipynb`, где студенты самостоятельно проходят этапы чтения данных, объединения таблиц, проверки качества, анализа распределений, корреляций и выбора признаков для будущих моделей.

Ноутбук намеренно содержит только задания и подсказки по инструментам. Рабочий код и выводы должны быть написаны студентами самостоятельно.

## Как выполнять и сдавать задания

Исходный ноутбук с заданием не нужно редактировать напрямую. Каждый студент создаёт свою копию ноутбука и добавляет к имени файла префикс со своим именем.

Пример:

```text
student_preprocessing_tasks.ipynb
ilyas_student_preprocessing_tasks.ipynb
```

Работу нужно выполнять в своём персональном ноутбуке. После завершения задания студент добавляет свой ноутбук в репозиторий и вливает изменения в ветку `main`.

## Структура

```text
data/
  gamma.csv
  proton.csv
Parameters_Definitions_Report.pdf
student_preprocessing_tasks.ipynb
```
