# TAIGA IACT Preprocessing Course

Учебный мини-проект по первичной подготовке данных TAIGA IACT для задачи классификации событий `gamma` и `proton`.

В репозитории собраны:

- CSV-данные с параметрами Хилласа для gamma- и proton-событий;
- `data/proton.csv` как подготовленная таблица proton-событий;
- PDF-справочник по параметрам TAIGA-IACT;
- учебный ноутбук `student_preprocessing_tasks.ipynb`, где студенты самостоятельно проходят этапы чтения данных, объединения таблиц, проверки качества, анализа распределений, корреляций и выбора признаков для будущих моделей.

Ноутбук намеренно содержит только задания и подсказки по инструментам. Рабочий код и выводы должны быть написаны студентами самостоятельно.

## Структура

```text
data/
  gamma.csv
  proton.csv
Parameters_Definitions_Report.pdf
student_preprocessing_tasks.ipynb
```

## Основная цель

К концу работы студент должен получить осмысленный список `selected_features`: набор признаков, пригодных для следующего этапа курса - обучения и сравнения моделей классификации.
